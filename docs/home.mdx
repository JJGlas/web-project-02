# Home

Welcome to a collection of comprehensive guides based on personal experience and observations, covering video and audio encoding to cutting edge formats such as AV1 and VVC through the Command Line Interface (CLI). These guides are designed to be beginner-friendly; once you have the tools installed, encoding can actually be simpler via the CLI than through a GUI. The guides are intended to inform you enough to get you started; they are not an entire wiki covering every parameter and aspect. All the tools and methods discussed are ones I personally use and highly recommend.
